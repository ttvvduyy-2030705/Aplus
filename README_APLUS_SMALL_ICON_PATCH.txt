Patch icon Aplus: logo đã được thu nhỏ còn khoảng 50–56% khung icon để tránh bị launcher Android phóng to/tràn viền.

Cách dùng:
1) Giải nén zip này vào thư mục gốc source Aplus-main và chọn replace/overwrite.
2) Android: cd android && ./gradlew clean
3) Gỡ app cũ khỏi máy/emulator rồi cài lại để xoá cache launcher icon.

Chỉ chứa file icon/logo, không đụng chức năng app.
