# Aplus Batch 04 Icon Map

Theo yêu cầu dùng icon theo hướng Flaticon, các vị trí icon mới trong Batch 04 đã được tách thành file PNG riêng để dễ thay bằng pack Flaticon của công ty nếu cần.

## File mới

- `credential.png` → Credential Hub / model quyền mở khóa chung
- `recipient.png` → UI-47 chọn người nhận quyền
- `capability.png` → UI-69 kiểm tra tương thích model khóa
- `revoked.png` → trạng thái thu hồi mềm / Revoked
- `sync.png` → Pending Sync / Pending Revoke
- `matrix.png` → Permission matrix / capability matrix

## Quy tắc thay icon

Giữ nguyên tên file PNG trong `src/assets/icons`. Nếu dùng icon tải từ Flaticon, chỉ cần thay file cùng tên và giữ kích thước PNG nền trong suốt.
