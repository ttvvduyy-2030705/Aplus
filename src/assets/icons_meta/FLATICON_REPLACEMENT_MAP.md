# Aplus Icon Replacement Map

Các màn Batch 05 dùng icon: password, pin, calendar, refresh, check, unlock, close, revoked, history, plus, chevron.

Nếu công ty có tài khoản Flaticon/Premium, tải các icon cùng chủ đề từ https://www.flaticon.com/ rồi thay PNG cùng tên trong `src/assets/icons/`:
- `password.png` -> password/keypad/pin code icon
- `pin.png` -> numeric pin icon
- `calendar.png` -> schedule/calendar icon
- `revoked.png` -> revoke/block/ban icon
- `sync.png` -> sync/cloud icon
- `refresh.png` -> random/regenerate icon

Lưu ý: nếu dùng Flaticon free cần attribution; Premium có thể dùng không cần attribution theo giấy phép tải từ tài khoản.
